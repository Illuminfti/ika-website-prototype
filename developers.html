Quick start

Define authority in a few lines

TypeScript 
// Define a cross-chain wallet with spending policy
const wallet = await ika.createWallet({
  owner: solanaPubkey,
  chains: ["bitcoin", "ethereum"],
  policy: {
    spendingLimit: { daily: "1000 USDC" },
    allowlist: ["uniswap.eth", "aave.eth"],
    requireUser: { above: "500 USDC" },
  }
});

// Sign a cross-chain transaction
const sig = await wallet.sign(ethTransaction);
